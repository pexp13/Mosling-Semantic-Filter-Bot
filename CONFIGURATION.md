# Настройка интересов и параметров

---

## Как описать свои интересы

Самая важная часть скрипта — блок `INTERESTS`. Именно его Gemini читает, чтобы решить, релевантно ли письмо.

**Принципы хорошего описания:**
- Пишите естественным языком, не только ключевые слова
- Упоминайте смежные области — это помогает находить косвенно связанные темы
- Можно писать на русском или английском (Gemini понимает оба)

---

## Пример настройки по умолчанию

```javascript
const INTERESTS = `
Мои научные интересы: анализ тональности, сентимент-анализ, динамический сентимент-анализ, 
динамический анализ тональности, эмоции в языке, аффективная лингвистика, 
субъективность в тексте, мнения и оценки в языке, эмоциональная окраска текста.

Смежные области, которые меня интересуют: прагматика, дискурс-анализ, 
корпусная лингвистика, обработка естественного языка (NLP), 
конфликтология текста, риторика, аргументация.
`;
```

---

## Как добавить больше тем

Просто дополните блок описанием новых направлений:

```javascript
const INTERESTS = `
[ваши основные интересы]

Дополнительно меня интересует: [новая область].
Также релевантны: [ещё одна область, конференции по X, гранты на Y].
`;
```

---

## Настройка частоты проверки

По умолчанию скрипт запускается **раз в час**. Если письма приходят редко, можно изменить на раз в 2–4 часа в настройках триггера — это снизит нагрузку на Gemini API.

---

## Изменение количества символов из письма

По умолчанию скрипт отправляет в Gemini первые **2000 символов** письма:

```javascript
const body = message.getPlainBody().substring(0, 2000);
```

Если письма длинные и важная информация может быть в конце — увеличьте до `4000`. Если хотите сэкономить запросы — уменьшите до `1000`. Для большинства объявлений 2000 символов достаточно.

---

## Изменение отправителя

Если вы хотите отслеживать другую рассылку — измените:

```javascript
const SENDER = "другой_адрес@example.com";
```

Можно также указать домен:

```javascript
const SENDER = "@googlegroups.com"; // все письма от Google Groups
```

---

## Добавление нескольких отправителей

Если нужно отслеживать несколько рассылок, замените строку поиска писем:

```javascript
// Вместо одного отправителя
const threads = GmailApp.search(`from:${SENDER} -label:${LABEL_NAME}`);

// Несколько отправителей
const threads = GmailApp.search(
  `(from:mosling@googlegroups.com OR from:другая@рассылка.com) -label:${LABEL_NAME}`
);
```
