# Установка и настройка

Следуйте шагам по порядку — это займёт около 10 минут.

---

## Шаг 1. Создать Telegram-бота

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду `/newbot`
3. Придумайте имя бота (например: `Mosling Notifier`)
4. Придумайте username (должен заканчиваться на `bot`, например: `mosling_notify_bot`)
5. BotFather выдаст вам **токен** — длинная строка вида `123456789:AAFxxxxxxx`
6. Сохраните токен

**Узнать свой chat_id:**
1. Найдите бота **@userinfobot**
2. Отправьте ему любое сообщение
3. Он ответит вашим `id` — это и есть `TG_CHAT_ID`

> ⚠️ Перед запуском скрипта обязательно напишите своему боту хотя бы одно сообщение — иначе он не сможет отправлять вам уведомления.

---

## Шаг 2. Получить Gemini API key

1. Перейдите на [aistudio.google.com](https://aistudio.google.com)
2. Войдите под своим Google-аккаунтом
3. Нажмите **"Get API key"** → **"Create API key"**
4. Скопируйте ключ и сохраните его

> Бесплатный план: 1500 запросов в день — более чем достаточно для рассылки.

---

## Шаг 3. Создать проект в Google Apps Script

1. Перейдите на [script.google.com](https://script.google.com)
2. Нажмите **"Новый проект"**
3. Переименуйте проект (например: `MoslingBot`)
4. Удалите весь код в редакторе
5. Вставьте содержимое файла [`src/main.js`](src/main.js)

---

## Шаг 4. Заполнить переменные

В начале скрипта найдите блок `НАСТРОЙКИ` и заполните четыре переменные:

```javascript
const GEMINI_API_KEY = "ВАШ_GEMINI_KEY";
const TG_BOT_TOKEN = "ВАШ_ТОКЕН_БОТА";
const TG_CHAT_ID = "ВАШ_CHAT_ID";
const SENDER = "mosling@googlegroups.com"; // не менять
```

Также отредактируйте блок `INTERESTS` — опишите свои научные интересы своими словами (см. [CONFIGURATION.md](CONFIGURATION.md)).

---

## Шаг 5. Запустить первый раз вручную

1. В редакторе выберите функцию `checkMoslingEmails` в выпадающем списке вверху
2. Нажмите **▶ Выполнить**
3. Google попросит разрешения — нажмите **"Просмотреть разрешения"** → выберите свой аккаунт → **"Разрешить"**
4. Скрипт запустится и проверит почту

> Это нужно сделать один раз для выдачи разрешений.

---

## Шаг 6. Настроить автозапуск (триггер)

1. В левом меню нажмите на иконку **часов** (Триггеры)
2. Нажмите **"Добавить триггер"** (внизу справа)
3. Настройте:
   - Функция: `checkMoslingEmails`
   - Источник событий: `По времени`
   - Тип триггера: `Таймер с интервалом в часах`
   - Интервал: `Каждый час`
4. Нажмите **"Сохранить"**

Готово! Теперь скрипт будет автоматически проверять почту каждый час.

---

## Проверка работы

Чтобы убедиться, что всё работает:
1. Найдите любое старое письмо от mosling в Gmail
2. Снимите с него метку `mosling_processed` (или удалите метку целиком в настройках Gmail)
3. Запустите скрипт вручную ещё раз
4. Если письмо релевантно — придёт уведомление в Telegram

---

## Возможные проблемы

| Проблема | Решение |
|---|---|
| Ошибка 403 от Gemini | Проверьте правильность API key |
| Telegram не отвечает | Убедитесь, что написали боту первое сообщение |
| Письма не находятся | Проверьте адрес `SENDER` и что письма действительно в Gmail |
| Дублирующиеся уведомления | Проверьте, что метка `mosling_processed` создалась в Gmail |
